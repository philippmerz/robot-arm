Description
The LCM-5MP-8MM is a high resolution (up to 5MP) lens designed for cameras with a 1/1.8" sensor or smaller, to create sharp images. The compact size and affordable price make this lens a very good choice for all standard vision applications. The LCM-5MP serie has multiple focal lengths to assure the right working distance can be met for each application. The iris and focus are adjustable and can be fixed by a screw.

Please note that this lens may produce some lens shading when used with a 1/1.8" sensor. If this is not acceptable we recommend the following lens: LCM-5MP-08MM-F1.4-1.5-ND1,
